估值方法学Valuation methodology

1 · 创新药 rNPV 模型1 · New drug rNPV model

rNPV = Σ (CF_t × PoS_t) / (1+r)^t - Σ Cost_t × PoS_t / (1+r)^t PoS_stage: preclin 5% · Ph1 10% · Ph2 17% · Ph3 52% · NDA 88% · approved 100% r (WACC): 11% · horizon: launch+10y LOE · terminal: 0 峰值销售到达 launch+5y · 爬坡曲线Peak sales reached at launch+5y · ramp curve 10/30/60/85/100/100/95/80/55/25

2 · 类似药利润分成模型2 · Biosimilar profit-share model

NPV = Σ (PeakSales × (1-Discount) × MarketShare_t × Margin × ProfitShare) / (1+r)^t 默认值: 分成 35-45% · 对原研折价 25-35% · 毛利 55% · 市占率爬坡Defaults: share 35-45% · discount vs originator 25-35% · GM 55% · share ramp 5/20/40/55/65/70

3 · 交易结构拆分3 · Deal structure decomposition

Upfront : Milestone : Royalty ≈ 8% : 40% : 52% (新药全球均值)(global new drug average) 按阶段首付占比:Upfront % by stage: preclin 3-5% / Ph1 6-10% / Ph2 10-15% / Ph3 15-25% royalty rate: Ph1 8-11% · Ph2 10-13% · Ph3 12-16% · Mkt 14-20%

4 · 买方匹配评分 (满分 100)4 · Buyer matching score (out of 100)

Score = 40·IndicationFit + 25·ModalityFit + 20·StageFit + 10·FinancialCapacity + 5·RecentActivity - IndicationFit: 命中 1.0 / 邻接 0.6 / 同治疗领域 0.3 / 否则 0 - ModalityFit: 平台成熟 1.0 / 有管线 0.7 / 开放态度 0.4 / 否则 0.1 - StageFit: 偏好窗口命中 1.0 / 相邻 0.6 - FinancialCapacity: log(cash+revenue) 归一化 - RecentActivity: 近12月BD次数归一化- IndicationFit: direct hit 1.0 / adjacent 0.6 / same TA 0.3 / else 0 - ModalityFit: core platform 1.0 / active pipeline 0.7 / open stance 0.4 / else 0.1 - StageFit: preferred stage 1.0 / adjacent 0.6 - FinancialCapacity: log(cash+revenue) normalized - RecentActivity: BD count last 12M, normalized
